# 💕 一起变轻 - 情侣减肥打卡 App

情侣专属的减肥打卡与互相鼓励 App，用轻量记录 + 情绪价值引擎，帮助两个人长期坚持并达成健康目标。

## ✨ 核心功能

### 1. 极简打卡（≤30秒）
- 最少选择1项即可完成打卡
- 支持补卡过去2天
- 字段包括：运动、饮食、喝水、睡眠、心情、备注、体重（可选）

### 2. AI情绪反馈
- 自动识别打卡状态（努力程度、心情状态、风险标识）
- 生成个性化反馈卡（共情 + 成就 + 小动作）
- 支持4种文案风格：可爱、克制、搞笑、认真

### 3. 情侣空间
- 双方今日打卡状态展示
- 情侣能量条（综合完成度，不排名）
- 共同目标进度条 + 里程碑庆祝（10%/50%/100%）
- 鼓励墙（夸夸/抱抱/小任务）

### 4. 进展追踪
- 体重趋势图（7/30/90天）
- 习惯热力图（打卡/运动/喝水/睡眠）
- 里程碑时间轴（7/14/30连续打卡）
- 统计卡片（连续打卡、总打卡次数、运动天数等）

### 5. 周报总结
- 自动生成本周周报
- 亮点总结、进步回顾、下周小目标
- 情侣互动时刻记录

### 6. 隐私与设置
- 字段级共享控制（体重/围度/照片/心情/备注）
- 提醒设置（每日打卡、伴侣打卡、周报）
- 情侣绑定/解绑

## 📱 使用流程

### 新手流程（≤60秒）

1. **登录**：输入手机号和昵称（首次自动注册）
2. **绑定伴侣**：创建或加入6位绑定码
3. **设置目标**：体重目标、记录强度、文案风格
4. **开始打卡**：选择日期，至少1项即可完成

### 每日打卡（≤30秒）

1. 选择日期（今天/昨天/前天）
2. 勾选完成项（运动/饮食/喝水/睡眠/心情）
3. 可选填写备注和体重
4. 提交后查看AI反馈卡
5. 一键发送鼓励给伴侣

当前版本使用 `localStorage` 存储数据，包括：
- 用户资料
- 打卡记录
- 情侣绑定
- 目标与里程碑
- 鼓励卡片
- 设置与隐私配置

## 🛠️ 技术栈

- **前端框架**：React 18 + TypeScript

## 🎨 设计原则

1. **不制造焦虑**：趋势为主，隐私可控
2. **情绪价值优先**：共情、鼓励、庆祝、恢复
3. **极简体验**：30秒完成打卡
4. **情侣私域**：默认仅双方可见

## 📝 注意事项

- 当前版本为MVP，数据存储在浏览器本地
- 建议定期导出数据备份
- 清除浏览器数据会导致数据丢失
- 多设备间数据不同步（需后端支持）
