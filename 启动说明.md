# 启动说明

## 开发服务器已启动

服务器正在运行在：**http://localhost:5173** 或 **http://127.0.0.1:5173**

## 如果遇到错误 -102，请尝试以下方法：

### 方法1：使用 127.0.0.1 而不是 localhost
在浏览器中访问：**http://127.0.0.1:5173**

### 方法2：清除浏览器缓存
1. 按 `Ctrl + Shift + Delete` 打开清除浏览数据
2. 选择"缓存的图片和文件"
3. 点击"清除数据"
4. 重新访问 http://localhost:5173

### 方法3：使用无痕模式
1. 按 `Ctrl + Shift + N` (Chrome) 或 `Ctrl + Shift + P` (Firefox)
2. 在无痕窗口中访问 http://localhost:5173

### 方法4：检查防火墙
确保 Windows 防火墙没有阻止 Node.js 或端口 5173

### 方法5：重启开发服务器
1. 在终端中按 `Ctrl + C` 停止服务器
2. 运行 `npm run dev` 重新启动

## 验证服务器是否运行

在浏览器中访问以下任一地址：
- http://localhost:5173
- http://127.0.0.1:5173

如果看到应用界面，说明服务器正常运行。

## 常见问题

**Q: 端口被占用怎么办？**
A: Vite 会自动尝试下一个可用端口（5174, 5175...），查看终端输出确认实际端口。

**Q: 修改代码后页面不更新？**
A: 检查浏览器控制台是否有错误，尝试硬刷新（Ctrl + F5）。

**Q: 仍然无法访问？**
A: 检查是否有其他程序占用端口，或尝试更改端口号。
