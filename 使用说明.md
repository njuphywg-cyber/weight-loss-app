# 🚀 快速开始 - 无需安装 Node.js

## ✨ 最简单的使用方法

我已经为你创建了一个**单文件版本**的应用，可以直接在浏览器中打开使用，**完全不需要安装 Node.js**！

## 📖 使用步骤

1. **找到文件**
   - 文件位置：`C:\Users\wag17423\weight-loss-app\index-standalone.html`
   - 或者直接双击这个文件

2. **打开文件**
   - 双击 `index-standalone.html` 文件
   - 或者右键选择"打开方式" → 选择浏览器（Chrome、Edge、Firefox 等）

3. **开始使用**
   - 应用会自动在浏览器中打开
   - 所有数据会保存在浏览器的本地存储中
   - 下次打开同一个浏览器，数据还在

## 🎯 功能说明

- **切换用户**：点击顶部的"我"或"女朋友"按钮切换用户
- **记录体重**：填写日期、体重和备注，点击"保存记录"
- **查看历史**：左侧显示所有历史记录，可以看到每次的体重变化
- **查看趋势**：右侧显示体重变化趋势图表

## 💾 数据存储

- 所有数据保存在浏览器的 **LocalStorage** 中
- 数据只保存在你使用的浏览器中
- 清除浏览器数据会删除所有记录（建议定期备份）

## 🔄 如果需要使用完整版本

如果你想使用需要 Node.js 的完整版本（可以自定义更多功能），可以：

1. 安装 Node.js（从 https://nodejs.org/ 下载）
2. 在项目目录运行：
   ```powershell
   npm install
   npm run dev
   ```

## ❓ 常见问题

**Q: 为什么图表显示不出来？**
A: 确保你的电脑可以访问互联网，因为图表库是从 CDN 加载的。

**Q: 数据会丢失吗？**
A: 数据保存在浏览器本地，只要不清除浏览器数据就不会丢失。建议定期记录数据。

**Q: 可以在手机上使用吗？**
A: 可以！你可以把这个 HTML 文件放到手机上，或者部署到服务器上。
